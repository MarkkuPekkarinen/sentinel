# Code of Conduct

Be respectful. Be constructive. Don't be a jerk.

## The short version

- Treat others the way you'd want to be treated
- Critique code, not people
- Assume good intent — most misunderstandings are just that
- Keep discussions technical and on-topic
- No spam, no harassment, no personal attacks

## Enforcement

If someone is making the project worse for everyone, maintainers will step in. Actions range from a warning to a permanent ban, depending on severity. If you have a problem, open an issue or email the maintainers directly.

That's it. We're here to build good software.
